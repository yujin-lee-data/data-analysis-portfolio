# 데이터 정의서

> 이 문서는 분석에 사용한 데이터 구조, 주요 컬럼과 해석 유의사항을 정리합니다.

## 1. 사용 데이터

| 항목 | 내용 |
|---|---|
| 데이터셋 | Google BigQuery Public Dataset GA4 Ecommerce Sample |
| 분석 성격 | 공개 샘플 데이터 기반 관찰 분석 |
| 분석 단위 | 익명 사용자, 이벤트, 상품 카테고리 |
| 기간 기준 | 데이터셋 내 이벤트 발생 기간 기준 |

## 2. 주요 필드

| 필드 | 의미 | 분석 활용 |
|---|---|---|
| `event_date` | 이벤트 발생일 | 일자별 활동 추이 확인 |
| `event_name` | 이벤트 이름 | 퍼널 단계 정의 |
| `user_pseudo_id` | 익명 사용자 식별자 | 사용자 수 기준 전환율 계산 |
| `device.category` | 디바이스 유형 | 디바이스별 전환 차이 확인 |
| `items.item_category` | 상품 카테고리 | 카테고리별 전환 차이 확인 |

## 3. 핵심 이벤트

| 이벤트 | 퍼널 단계 | 의미 |
|---|---:|---|
| `session_start` | 1 | 세션 시작 |
| `view_item` | 2 | 상품 상세 조회 |
| `add_to_cart` | 3 | 장바구니 추가 |
| `begin_checkout` | 4 | 결제 시작 |
| `purchase` | 5 | 구매 완료 |

## 4. 데이터 사용상 주의사항

본 데이터는 Google BigQuery에서 제공하는 공개 샘플 데이터이며 실제 기업 내부 데이터가 아닙니다. 데이터 기준 기간은 프로젝트 수행 기간이 아니라 데이터셋 내 이벤트 발생 기간입니다. 이벤트 파라미터와 item 정보는 반복 구조로 저장될 수 있으므로 SQL 작성 시 UNNEST 처리 기준을 일관되게 적용했습니다.
