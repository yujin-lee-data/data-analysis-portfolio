# 지표 정의서

> 이 문서는 분석에 사용한 핵심 지표와 계산·해석 기준을 정리합니다.

## 1. 퍼널 단계

| 단계 | 이벤트 | 의미 |
|---:|---|---|
| 1 | `session_start` | 세션 시작 |
| 2 | `view_item` | 상품 상세 조회 |
| 3 | `add_to_cart` | 장바구니 추가 |
| 4 | `begin_checkout` | 결제 시작 |
| 5 | `purchase` | 구매 완료 |

## 2. 핵심 지표

| 지표 | 계산 기준 | 해석 |
|---|---|---|
| 단계별 사용자 수 | 각 이벤트를 수행한 고유 `user_pseudo_id` 수 | 단계별 도달 규모 확인 |
| 직전 단계 대비 전환율 | 현재 단계 사용자 수 / 직전 단계 사용자 수 | 바로 이전 단계에서 다음 단계로 이동한 비율 |
| 직전 단계 대비 이탈률 | 1 - 직전 단계 대비 전환율 | 전환이 약해지는 구간 확인 |
| 시작 단계 대비 누적 전환율 | 현재 단계 사용자 수 / `session_start` 사용자 수 | 최초 유입 대비 최종 도달 수준 확인 |
| 우선 점검 구간 | 이탈 규모와 이탈률을 함께 고려한 구간 | 서비스 개선 검토가 필요한 후보 구간 확인 |

## 3. 해석 유의사항

본 프로젝트의 퍼널은 분석자가 정의한 대표 행동 흐름입니다. 모든 사용자 여정을 완벽히 재현하는 것은 아니므로 결과 해석 시 세션 단위 순서, 이벤트 파라미터 적재 범위, 사용자 기준 집계와 이벤트 기준 집계의 차이를 함께 고려해야 합니다.
