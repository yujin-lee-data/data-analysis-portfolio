# 데이터 정의서

> 이 문서는 분석에 사용한 데이터 구조, 주요 컬럼과 해석 유의사항을 정리합니다.

## 1. 사용 데이터

| 파일 | 내용 | 분석 활용 |
|---|---|---|
| `transaction_data.csv` | 고객별 장바구니, 상품, 매출, 수량, 할인, 상대 기간 정보 | 매출·재구매·쿠폰 사용 분석 |
| `product.csv` | 상품 ID별 부서, 브랜드, 상품군 정보 | 상품군별 매출 구조 분석 |
| `campaign_table.csv` | 캠페인별 대상 고객 정보 | 캠페인 대상 고객 식별 |
| `campaign_desc.csv` | 캠페인 유형과 시작·종료 DAY 정보 | 캠페인 유형별 반응 비교 |
| `coupon_redempt.csv` | 고객별 쿠폰 사용 정보 | 쿠폰 사용 여부 분석 |
| `hh_demographic.csv` | 일부 고객의 인구통계 정보 | 고객군 보조 분석 |

## 2. 주요 컬럼

| 컬럼 | 의미 | 분석 활용 |
|---|---|---|
| `household_key` | 고객 식별자 | 고객 단위 집계 |
| `BASKET_ID` | 장바구니/거래 식별자 | 거래 단위 집계 |
| `PRODUCT_ID` | 상품 식별자 | 상품 정보 연결 |
| `SALES_VALUE` | 거래 라인의 매출값 | 매출 규모 계산 |
| `QUANTITY` | 구매 수량 | 구매량 확인 |
| `DAY` | 데이터셋 내부 상대 일자 | 기간 흐름 확인 |
| `WEEK_NO` | 데이터셋 내부 상대 주차 | 주차별 매출 추이 확인 |
| `COUPON_DISC`, `COUPON_MATCH_DISC` | 쿠폰 할인 관련 값 | 쿠폰 사용 여부 및 할인 규모 확인 |

## 3. 데이터 사용상 주의사항

DAY와 WEEK_NO는 실제 달력 날짜가 아닙니다. 분석 기간은 공개 데이터셋 내 이벤트 발생 범위를 의미합니다. 쿠폰·캠페인 분석은 실험군/대조군 기반의 인과 분석이 아니라 관찰 데이터 기준 반응 차이로 해석합니다.
