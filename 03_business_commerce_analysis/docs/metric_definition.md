# 지표 정의서

> 이 문서는 분석에 사용한 핵심 지표와 계산·해석 기준을 정리합니다.

## 1. 핵심 지표

| 지표 | 계산 기준 | 해석 |
|---|---|---|
| 총 매출값 | `SALES_VALUE` 합계 | 전체 거래 규모 확인 |
| 고객 수 | `household_key` 고유 개수 | 분석 대상 고객 규모 확인 |
| 장바구니 수 | `BASKET_ID` 고유 개수 | 거래 발생 규모 확인 |
| 평균 장바구니 금액 | 장바구니별 `SALES_VALUE` 합계의 평균 | 거래당 평균 구매 규모 확인 |
| 재구매 고객 | 장바구니 수가 2개 이상인 고객 | 반복 구매 행동 확인 |
| 매출 기여 고객군 | 고객별 총 매출 기준 상위 20%, 중간 60%, 하위 20% | 고객군별 매출 집중도 확인 |
| 상품군 매출 | `COMMODITY_DESC`별 `SALES_VALUE` 합계 | 상품군별 매출 기여도 확인 |
| 쿠폰 장바구니 비율 | 쿠폰 할인 금액이 0보다 큰 장바구니 비율 | 쿠폰 사용 거래 비중 확인 |
| 캠페인 관찰 응답률 | 캠페인 대상 고객 중 쿠폰 사용 고객 비율 | 캠페인별 관찰상 반응 차이 확인 |

## 2. 해석 유의사항

캠페인 관찰 응답률은 쿠폰 사용 여부를 집계한 값입니다. 실험군/대조군 기반 A/B 테스트가 아니므로 매출 증가 원인으로 단정하지 않습니다. 본 지표는 공개 데이터 기준의 고객·매출 구조와 반응 차이를 정리하기 위한 참고 지표입니다.
